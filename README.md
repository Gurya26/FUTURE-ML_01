# Sales Forecasting using Machine Learning

## Project Overview

This project focuses on predicting future sales using historical business data.
A Linear Regression model is used to forecast sales trends and analyze business performance.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Steps Performed

* Data loading and preprocessing
* Time-based feature engineering (Days, DayOfWeek, Month)
* Model training using Linear Regression
* Model evaluation using MAE (Mean Absolute Error)
* Future sales prediction
* Data visualization using bar charts

## Key Insights

* Sales show a clear increasing trend over time
* Peak sales observed in November
* Later months outperform early months
* Model captures overall trend effectively

## Output

* Monthly Sales Visualization (Bar Graph)
* Future Sales Predictions

## Conclusion

This project demonstrates how machine learning can be applied to real-world business problems like sales forecasting and decision-making.
